Fantasy Prompt Generator
「Agasa Noble Art」がお届けする、AI画像生成のためのファンタジープロンプトジェネレーターです。
直感的な操作で、高品質なファンタジーアートや塗り絵用線画のプロンプトを生成できます。

使い方
ブラウザで index.html を開きます。

画面上のボタン（世界観、キャラクター、背景など）を選択します。

生成されたプロンプトの下にある「コピー」ボタンを押して、お使いのAI生成ツール（Google Flowなど）に貼り付けてください。
